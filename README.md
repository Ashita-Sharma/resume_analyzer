# AI Resume Matcher 

An AI-powered web application that analyzes how well a resume matches a job description using a hybrid Natural Language Processing (NLP) approach.

The system combines semantic similarity, keyword matching, and skill extraction to generate an interpretable match score along with actionable feedback.
